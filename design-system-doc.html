Color System / 01

Every color, and when to use it.

A dominant dark background, a single violet accent family, and three supporting accent colors. Never use more than one accent at a time in the same component.

Typography / 02

Three fonts. Three roles. Never mix them up.

Each font has a strict job. Syne for structure and impact. Fraunces for emotion and contrast. DM Mono for everything else — data, body, UI text.

Display / Headlines
Syne · 700 / 800
Designing
Systems.
Used for: Hero titles, section titles, decision numbers, metric numbers. Always tight letter-spacing (-0.02em to -0.04em).
Italic / Emphasis
Fraunces · 300 italic
not a tool — a workflow.
Used for: Subheadings under Syne titles, decision headlines, hero subtitles. Creates visual contrast and emotional weight through serif + italic contrast.
Body / UI / Code
DM Mono · 300 / 400 / 500
Enterprise teams translating documents face
→ inconsistent terminology
→ no batch control
→ unclear cost estimation
Used for: Body copy, labels, badges, code blocks, metadata, captions, nav items. Monospace gives a precise, technical, audit-ready feel appropriate for enterprise SaaS.
Labels / Eyebrows
DM Mono · letter-spacing: 0.15–0.25em · uppercase
Design Decisions · / 03
Used for: Section labels, eyebrow text above headings, badge text, nav links. Always uppercase, always spaced out, always small (10–12px).

Visual Effects / 05

The invisible details that make it feel premium.

Three layered effects that create depth and atmosphere. Each one is subtle alone — together they define the aesthetic.

Grain Texture Overlay
Applied to the entire page via a fixed pseudo-element. Creates organic, anti-digital texture that makes the dark background feel handcrafted rather than digital.
body::before {
  content: '';
  position: fixed; inset: 0;
  background-image: url("data:image/svg+xml,
    ...feTurbulence baseFrequency='0.9'...");
  pointer-events: none;
  z-index: 1000;
  opacity: 0.6;
}
Radial Gradient Hero BG
A large violet radial gradient sits behind hero sections, giving the impression of a distant light source. Keeps the dark background from feeling flat.
.hero-bg {
  background:
    radial-gradient(
      ellipse 80% 60% at 10% 50%,
      rgba(124,58,237,0.12) 0%,
      transparent 60%
    );
}
Gradient Text
Applied to the most important word or line in a hero or section title. Creates a luminous, memorable highlight without color overload.
.gradient-text {
  background: linear-gradient(
    135deg,
    #a855f7 0%,
    #c084fc 50%,
    #a855f7 100%
  );
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}
Nav Blur / Frosted Glass
The sticky navigation uses backdrop-filter to blur content beneath it. Creates layering depth without heavy shadows.
nav {
  position: sticky; top: 0;
  background: rgba(10,10,15,0.8);
  backdrop-filter: blur(20px);
  -webkit-backdrop-filter: blur(20px);
  border-bottom: 1px solid var(--line);
}
1px Grid Divider Trick
Instead of adding individual borders to grid children, set gap:1px and background:var(--line) on the parent. The gap color becomes the border.
.grid-parent {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 1px;
  background: var(--line); /* ← border */
  border: 1px solid var(--line);
  overflow: hidden;
}
.grid-child {
  background: var(--bg2); /* ← fill */
}
Eyebrow Line Decoration
A short horizontal line before eyebrow/label text. Built with a ::before pseudo-element. Adds rhythm and visual hierarchy without extra HTML.
.eyebrow {
  display: flex;
  align-items: center;
  gap: 16px;
}
.eyebrow::before {
  content: '';
  width: 48px; height: 1px;
  background: var(--violet-bright);
  flex-shrink: 0;
}
Motion Design / 06

Motion rules that feel intentional.

Every animation serves a purpose: revealing content, confirming interaction, or creating hierarchy. Never animate for decoration.

Fade Up (entry)
opacity 0→1 + translateY 24px→0
Used for cards entering viewport on scroll.
Staggered Hero
animation-delay: 0.1s, 0.2s, 0.3s, 0.45s
Hero title lines animate in sequence on page load.
Duration
0.5s – 0.7s
— standard —
Long enough to feel intentional, short enough to feel snappy.
Easing
ease (CSS default)
— smooth —
No cubic-bezier complexity needed. ease-in-out for emphasis.
Hover Transitions
0.2s – 0.3s
— instant feel —
Color and background changes on hover. Never 0s (jarring) or 0.5s+ (laggy).
Scroll Observer
IntersectionObserver · threshold: 0.1
— on viewport —
Trigger fadeUp when card enters viewport. No libraries needed.
Core Animation CSS
/* The only keyframe you need */ @keyframes fadeUp { from { opacity: 0; transform: translateY(24px); } to { opacity: 1; transform: translateY(0); } } /* Staggered hero text */ .line1 { animation: fadeUp 0.7s 0.1s ease both; } .line2 { animation: fadeUp 0.7s 0.2s ease both; } .line3 { animation: fadeUp 0.7s 0.3s ease both; } /* Scroll-triggered (JavaScript) */ const observer = new IntersectionObserver((entries) => { entries.forEach(entry => { if (entry.isIntersecting) { entry.target.style.opacity = '1'; entry.target.style.transform = 'translateY(0)'; } }); }, { threshold: 0.1, rootMargin: '0px 0px -60px 0px' }); cards.forEach(card => { card.style.opacity = '0'; card.style.transform = 'translateY(24px)'; card.style.transition = 'opacity 0.55s ease, transform 0.55s ease'; observer.observe(card); });
